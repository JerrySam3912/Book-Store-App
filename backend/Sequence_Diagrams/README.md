# Sequence Diagrams

This folder contains sequence diagrams for the Book Store application.

## Files

- `01_Login_Functionality.puml` - PlantUML format
- `01_Login_Functionality.mmd` - Mermaid format

## How to Use

### PlantUML
1. Copy code from `.puml` file
2. Paste into https://plantuml.com/plantuml/uml/
3. Download PNG/SVG

### Mermaid
1. Copy code from `.mmd` file
2. Paste into https://mermaid.live/
3. Download PNG/SVG

Or use in Markdown:
```markdown
```mermaid
[paste code here]
```
```
